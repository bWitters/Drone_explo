📊 ARCHITECTURE
================================================================================

LeaderStateMachine:
  ┌────────────┐
  │  TAKEOFF   │ (décollage 0-99 steps)
  └─────┬──────┘
        │ (après 100 steps)
        ▼
  ┌──────────────────────┐
  │  MOVING_FORWARD      │ (suit le gap)
  └──┬───┬──────────┬───┘
     │   │          │
     │   │  angle   └── (alignement si angle > 10°)
     │   │  > 10°       │
     │   │              ▼
     │   │          ┌───────────┐
     │   │          │ ALIGNMENT │
     │   │          └─────┬─────┘
     │   │                │ (aligné)
     │   │                └──────┘
     │   │
   distance > max
     │
     ▼
  ┌──────────────────────┐
  │ WAITING_FOR_FOLLOWER │ (attend le follower)
  └──────────┬───────────┘
             │ (distance ≤ max)
             └─────────────┘

FollowerStateMachine:
  ┌────────────┐
  │  TAKEOFF   │ (décollage 0-99 steps)
  └─────┬──────┘
        │ (après 100 steps)
        ▼
  ┌─────────────────────┐
  │ WAITING_FOR_MESSAGE │ (attend "Come closer")
  └──────┬──────────────┘
         │ (reçoit message)
         ▼
  ┌──────────────────┐
  │ GETTING_CLOSER   │ (distance diminue)
  └──┬────────────┬──┘
     │            │
     │ distance   │ distance
     │ ≤ min     │ > max
     │            │
     ▼            │
  ┌──────────┐    │
  │ FOLLOWING│◄───┘
  └──────────┘ (maintient distance stable)

💡 CONCEPTS CLÉS
================================================================================

State Machine (Machine à États Finie):
  - Ensemble fini d'états possibles
  - Transitions entre états (avec conditions optionnelles)
  - Callbacks (before/after) exécutés lors des transitions

Librairie 'transitions':
  - Légère et simple (~50KB)
  - Supports: conditions, callbacks, inheritance
  - Performance: identique aux if/else pour la même logique

États du Leader:
  TAKEOFF → MOVING_FORWARD ↔ WAITING_FOR_FOLLOWER
                    ↕
                ALIGNMENT

États du Follower:
  TAKEOFF → WAITING_FOR_MESSAGE → GETTING_CLOSER ↔ FOLLOWING
